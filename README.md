# Dynamic Asset Group — Prototype

Prototype for the **Asset Groups** initiative in Rokt One Platform.

## What is this?

Asset Groups replace the one-at-a-time Creative model with a **pool-based approach** — advertisers upload multiple headlines, body copies, CTAs, and images per group, and Rokt's algorithm assembles and serves the best-performing combinations automatically.

Inspired by Google Performance Max Asset Groups and Meta Advantage+ Creative.

## Key features shown

- **Asset group sidebar** — multiple groups per campaign, each with a theme/audience
- **Copy assets** — add up to 5 headlines, 3 body copies, 3 CTAs as individual chips
- **Visual assets** — upload up to 5 images per group
- **Asset Strength indicator** — live score + checklist (like Google PMax)
- **Combinations counter** — shows how many permutations are possible (H × B × C × Img)
- **Combination preview** — cycle through all possible combinations per format (Standard, Block, Tile, Hero)
- **Asset tags** — preview labels which headline/body/image/CTA each combination uses
